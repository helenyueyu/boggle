In this game, I assume that Y is a consonant. However, this can be tweaked if Y is considered a vowel.

Boggle scoring: Word-length - points

1) 3,4 - 1
2) 5 - 2
3) 6 - 3
4) 7 - 5
5) 8+ - 11
